3：整体气氛要求

所谓痛车，多多少少会涉及整体的风格差异，软萌的、帅气的、以及社交恐怖份子的。

而，我的风格是偏向“工业风+性冷淡+工具感”，我希望我给人的感觉不是攻击性或者二次元死宅风，而更多的是那种雷蛇→冰山美人的感觉（或者雷蛇的人设类似 外冷内热）。

4.1总结

很痴迷，那种那种无机质下的冰冷工具感，但是又透着淡淡的人文关怀。

没有出色的剧情演出，没有燃气的口号台词，装甲核心整个游戏给你带来的似乎只有一股冷到骨子里的无稽之感。
但是啊，作为核心来驱动着冰冷金属躯体的，却是一股熊熊燃烧的萝卜之魂。
那若隐若现的机械细节在爱好者眼里就如同美女不经意间展示的福利，光是这一点就足别让人血脉喷张。

4.2 痛车角色的设定总结报告

4.2.1心理报告：雷蛇

参考资料：明日方舟雷蛇WIKI

个人建议：参考资料很重要，很重要，很重要……….

4.2.1.1 基本信息

  • 姓名: 雷蛇
  • 年龄: 不详
  • 来源地: 偏远矿业小镇
  • 现隶属单位: 黑钢国际

4.2.1.2 概述

雷蛇是一名从偏远矿业小镇走出的少女，目前在黑钢国际工作。她的职业生涯和行为方式给人留下了深刻印象，尤其是她一板一眼的工作态度及冷静、有序的战斗风格。

4.2.1.3 工作态度与行为特征

雷蛇做事一丝不苟，严格按照既定计划执行任务，展现出优等生般的勤勉和可靠。在同事们眼中，她是一位非常踏实、可信赖的伙伴，充满了专业主义色彩。她的战斗总是保持着稳步的节奏：

  1. 攻击
  2. 推进
  3. 确认战地情况
  4. 与同伴及时交流
  5. 考虑战术
  6. 进行重复操作

这套流程化的作战模式让她显得如同机器一般——冷静而高效。

4.2.1.4 人际关系与交流

尽管雷蛇的工作态度近乎机械，但她与同事的关系却并非冰冷无情。她会主动与同伴交流战地情况，并在适当时机分享战术考量。从她的台词中可以感受到一股浓厚的官话气息，非常正经。然而，这并不意味着她缺乏情感或自私自利。

4.2.1.5 内心动机分析

如果仅凭表面观察，很容易将雷蛇误认为一个没有感情的精致利己主义者。然而，通过深入探究她的背景故事资料（特别是背景故事资料4），我们发现她内心深处隐藏着一种深沉的情感——对故乡的热爱与守护之情。她希望通过自己的努力，用手中的武器来保护家园，这种情感赋予了她冰冷外表下最具人文关怀的光辉。

4.2.1.6 人物干员的相关梗

外号：
黑钢国际相声团(与芙兰卡联用)——雷蛇和芙兰卡虽然都是黑钢国际安保公司的员工，但两人在加入黑钢之前是没有关系的。是因为后来雷蛇的上级给雷蛇派遣了一名搭档，这个人就是芙兰卡，两个人才有了交集，但两个人的性格由于不合(或者说是某种意义上的合得来吧)，常常拌嘴吵架，这点在游戏剧情中常有体现，雷蛇觉得芙兰卡太冲动，不够冷静；而芙兰卡认为雷蛇太严肃，过于谨慎。久而久之，这对搭档被人get到了萌点，取名相声团。

4.2.2 结论

雷蛇是一位看似冷静、严肃的人物，但其内心却充满了对故乡的情感与责任感。她的行为模式和工作态度虽带有高度的机械性，却并非毫无情感，而是源于对保护家乡的坚定信念。这种矛盾与统一，使她成为一个复杂而真实的个体，值得我们更深入的理解与尊重。

5.4.1 个人技能树

以下是您的技能树描述：

电子设计与仿真

  • CAD快捷看图、编辑助手：用于快速查看和编辑CAD文件的软件。
  • Altium Designer：综合性的电路设计自动化软件，用于PCB设计和布局。
  • LTspice：用于模拟电路的SPICE仿真器。
  • PSpice：另一款用于电路仿真的软件。
  • OrCAD Capture CIS：电路原理图设计工具。
  • Tina-TI：德州仪器提供的电路仿真工具。
  • DFM（Design For Manufacturing）：面向制造的设计软件，华秋DFM用于检查并优化设计，确保制造可行。

系统级设计

  • Ansys Electronic Desktop (EDT)：Ansys电子设计仿真平台。
  • Ansys EMIT：用于电磁干扰和兼容性分析的工具。
  • Ansys Savant：用于高频电磁场仿真的软件。
  • Ansys SIwave：信号完整性、功率完整性和EMI分析工具。
  • Ansys Twin Builder：系统级数字孪生建模和仿真平台。

嵌入式系统开发

  • STM32CubeMX：STMicroelectronics提供的配置和初始化STM32微控制器的工具。
  • Keil uVision：嵌入式软件开发的集成开发环境（IDE）。
  • Xilinx PlanAhead和ISE Design Suite：用于FPGA开发的工具。
  • Quartus II：Intel FPGA的开发软件。
  • TimeGen：时序图生成软件。
  • UART Assistant：串口调试工具。
  • CANalyzer：用于CAN总线分析的工具。

功率与能源管理

  • Power Stage Designer：TI公司提供的电源设计工具。
  • Filter Solutions：滤波器设计软件。
  • SCP Configurator：电源管理和配置工具。
  • LTpowerCAD II：Linear Technology的电源设计工具。

编程与开发

  • Matlab：数学计算和仿真工具。
  • Visual C++ 6.0：微软的C++开发环境。
  • Qt Creator：跨平台C++应用开发框架。
  • Notepad++：高级文本编辑器。
  • JSONedit：JSON文件编辑器。
  • ChatGPT：语言模型应用。

工程管理与协作

  • Github Desktop：版本控制和协作平台。
  • VMware Workstation：虚拟机管理工具。
  • FileZilla Client：FTP客户端。
  • NetAssistant：网络辅助工具。
  • EasyConnect：远程连接工具。
  • 力控：工控监控软件。

杂项工具

  • Snail EDA：EDA工具。
  • Apabi：电子书阅读器。
  • ZCANPRO：Zigbee CAN协议分析软件。

综合应用

  • SOLIDWORKS 2022：三维机械设计软件。
  • SIMetrix/SIMPLIS：混合信号仿真工具。
  • Mathcad 15：数学公式编辑和计算软件。

以上描述展示了在电子设计、系统仿真、嵌入式开发、功率管理、编程开发以及工程管理等多个领域的技能
