正面影响 最容易想到的就是，距离的阻碍消除了。过去，如果想要与远方的家人朋友联系，我们只能选择亲自去他们所在的地方，或者用邮寄书信的方式往来。但现在，无论在生活还是工作中，我们都可以实现异地交流。在生活中，我们可以异地聊天；在工作中，我们可以开远程会议。
其次，沟通的时效性增强了。过去，我们沟通要依赖写信。但现在有了微信、QQ等即时通讯软件后，我们可以几乎无延迟地接收和发送信息。
最后，是沟通渠道变得多元化。这个可以从两方面来说。第一，交友途径变多了，我们可以使用多种社交软件建立社交关系，比如微信（WeChat）、豆瓣（Dou ban）、微博（Weibo）等。第二，沟通形式变得丰富了，我们可以借助多种信息载体，比如文字、音频、视频、图片等来表达我们的想法和情感。
三大维度：距离，时效，渠道

反面影响 过度沉迷于网络世界 → 容易使人封闭在自己的空间 → 人们的人际交流能力下降 → 在真实的社交环境中感到不适应、出现交流障碍

从集体或社会层面来看，当大家都对科技上瘾时，就可能会降低人们之间相处的质量。比如：
朋友聚会时，大家都只顾着各自刷手机 → 缺乏人与人之间的真实沟通
过于依赖和信任网上的信息，不能辨别虚假信息 → 造成虚假信息的传播 → 引起误解 → 破坏人们之间的信任 → 阻碍人际关系的发展

影响 具体内容
正面影响 最容易想到肯定还是时空上的便利性。比如：
1：互联网 + 沟通协作软件 → 可以让人们灵活选择办公地点，并且实时沟通
除此之外，还有工具的升级。比如：
2：办公软件，如Excel的出现 → 轻松计算并整理数据 → 提升工作效率：
除了上面提到的，科技发展带来另一个新趋势就是任务分配方式的改变。比如：
3：机器人完成机械化、流程化、危险系数高的工作 → 把人从体力劳动中解放出来 → 人们去做更高级的工作
4：人工智能（AI）能够快速从大批量文档中提取关键信息并分类 → 大大减少人工处理信息的时间 → 人们去做更高级的工作
反面影响 首先，最直接的就是，工作中过度使用科技会危害身心健康，进而降低工作质量和效率。比如：
1：长时间使用计算机或电子设备 → 过长的屏幕使用时间 → 过度用眼，视力下降 → 更容易读错信息，造成工作上的失误，工作质量降低；或因为需要花更多精力读取信息，从而工作效率降低
2：长时间在电脑前工作，久坐不起 → 肥胖、颈椎病等健康问题 → 无法全身心投入工作，甚至需要休长时间的病假 → 影响工作
其次，使用过程中可能存在信息安全隐患。比如：
3：使用互联网 → 被黑客入侵，信息泄露 → 财产损失

科技与贫富差距

影响 具体内容
缩小差距 首先，科技的发展为所有人，包括贫困的群体，都带来了更多积累财富的机会。比如
1：网络商业平台的出现 → 提供更多就业和创业机会 → 给人们带来更多收入来源 → 缩小贫富差距其次，
科技带来了更多教育资源。比如
2：线上教育资源 → 让人们不受地域和空间的限制，都能接受教育 → 更多人能在线上学到知识和技能 → 获得更好的就业机会 → 缩小贫富差距
3：线上公开课程 → 受教育的成本降低，机会增多 → 更好的就业机会 → 缩小贫富差距
最后，科技的发展使得生活成本降低，比如：
4：科技提高生产效率 → 生活必需品的生产成本下降 → 生活必需品的价格降低 → 对贫困人群来说，更容易解决温饱问题，满足基本的生活需求 → 缩小贫富差距
综合上面这些情况来看，我们可以说，科技缩小了贫富差距。
扩大差距 上一部分讲到，科技影响了人们赚钱的机会和能力。但辩证地来看，这一现象也有可能会造成贫富差距的扩大，使得富人更富，穷人更穷。
首先，富人更富，具体来说，可能是：富人通常都拥有不可替代的技术、知识、能力，而这些正是推动科技发展的要素。因此，科技越是发展，这些人越是获益。
而穷人更穷，可能是因为：他们往往都缺乏赚钱的机会或者能力，所以通常从事重复性或技术含量更低的工作。然而，随着科技的发展，很多这样的工作都被机器所取代，这些穷人就会失业。更具体的例子可以是：人脸识别技术的广泛应用，可能会替代传统的门卫，这样门卫就可能会失业了。
因此，从这些角度来看，科技也可能会令贫富差距进一步扩大。

思路：工作满意度

从正面论证：

  • 金钱因素

    • 如果有高薪资和福利 → 就能养活自己、养家、实现经济独立 → 工作满意度提高

  • 工作环境

    • 如果有舒适的办公环境 → 就能心情愉悦、工作效率高 → 工作满意度提高

  • 晋升空间

    • 如果有晋升机会 → 就能知道自己的努力是有目标、有结果的 → 提升工作动力 → 工作满意度提高

  • 工作氛围

    • 如果同级之间互相尊重、上司提供支持 → 就能在工作时有归属感 → 工作满意度提高

  • 自我实现空间

    • 如果工作符合个人兴趣/有创造空间 → 就能激发灵感，乐于接受挑战 → 提高工作自主性 → 工作满意度提高

从反面论证：

  • 工作时长

    • 如果工作时间过久 → 就过于疲惫、工作效率低 → 没有work-life balance（工作与生活的平衡） → 工作满意度降低

  • 通勤时间

    • 如果通勤时间过长 → 就浪费时间、无意义地消耗精力 → 工作满意度降低

真思路

  1. 提出中心句：“我认为XXX很重要。”
  2. 解释情景：解释这个论点成立的具体情景。
  3. 正面推理或反面推理：正面：如果有XXX，就能带来什么好处；反面：如果没有XXX，就会造成什么不好的后果。

这三步能够帮助你搭建一个“证明某事很重要”的完整逻辑框架。我们分别用两个例子来看一下怎么应用。

  1. 中心句：我认为面试表现非常重要。
  2. 解释情景：许多候选人的基础条件都非常优秀，面试是进一步区分候选人的过程。
  3. 正面推理：如果有优异的面试表现，就能从相似条件的人中脱颖而出。
  1. 中心句：我认为大学学历对于找工作很重要。
  2. 解释情景：大学学历是许多公司招聘的门槛。
  3. 反面推理：如果没有大学学历，就会在简历筛选时被淘汰，就不能找到合适的工作。

TEXT2：6→犯罪

思路

思路：犯罪的原因与预防

所谓内在原因，就是人/物自身产生的因素，而外在原因指的是外界环境产生的因素。利用这个框架，我们可以把导致一个人犯罪的原因归类：

  • 内在原因

    • 性格问题：个体性格孤僻、共情力低下、反社会人格等
    • 法律意识淡薄：个体法律知识少、藐视法律等

  • 外在原因

    • 人际关系：与周围人关系不佳或有矛盾、被周围人教唆犯罪
    • 社会原因：遭受社会不公正的对待、对某种社会现象不满
    • 经济原因：经济形势下滑，导致贫困、失业、遭遇经济变故而难以生存
    • 法律体系漏洞：立法执法环节中有监管盲区，被人钻空子

我们来拿一道真题看一下，对具体的题目，我们应该怎么回答。

Many ex-prisoners continue to commit crimes even after being punished for it. Why do you think this happens? How can we solve this problem?

题目问：“刑满释放人员在受到惩罚后仍继续犯罪。为什么会出现这种情况？怎么解决这个问题?”

这道题的犯罪主体就是非常具体的“ex-prisoners”，前科犯。他们与初次犯罪的人最大的区别就在于，他们已经受过了刑罚，但是依然再次犯罪。我们按照前面列出的主观和客观原因来思考，并把它们具象化到具体的场景之中。

拿这道题来说，出狱后的罪犯面临的最大问题可能是如何重新融入社会，所以我们可以围绕这一点来展开。因此，他们再次犯罪的原因可能是：

  • 内在原因

    • 性格问题：报复心重 → 惩罚加重了对社会的仇视 → 再犯罪
    • 思想观念：在狱中并未完全被改造 → 再犯罪

  • 外在原因

    • 人际关系：入狱导致家人、朋友等的离散 → 社会关系破裂 → 再犯罪
    • 社会原因：社会对罪犯有偏见 → 受到歧视 → 产生仇恨敌对情绪 → 再犯罪
    • 经济原因：受到歧视 → 找不到工作 → 缺乏经济来源，无法支持生活 → 再犯罪
    • 法律体系漏洞：出狱后缺乏监管 → 再犯罪

那么，针对这些原因，我们就可以提出相应的解决措施。

  • 针对内在原因

    • 针对报复心重 → 执法部门提高执法能力，增强法律的威慑效果 → 防止再犯罪
    • 针对没有被完全改造 → 执法部门加强服刑人员思想教育 → 防止再犯罪

  • 针对外在原因

    • 针对社会关系破裂 → 政府建立出狱人士支持组织，提供社交机会和心理咨询服务 → 增强社会归属感 → 防止再犯罪
    • 针对受到社会偏见 → 政府加强社会宣传工作，减轻大众对出狱人员的抵触 → 有助于出狱人员回归社会 → 防止再犯罪
    • 针对找不到工作 → 政府在狱中开展职业技能教育，并提供出狱人员工作机会，保障其基本生活 → 防止再犯罪
    • 针对缺乏有效监管 → 执法部门建立完善严格的追踪系统 → 防止再犯罪

到这里，我们已经发散出了很多论点，你可以自己动笔写写消化一下这部分内容。

TEXT2：8→健康类

思路

考点1：造成健康问题的原因和解决措施

造成健康问题的原因：主观和客观

在讲原因时，我们可以从主观和客观两个维度来分析。这个方法，我们之前在《Task 2 犯罪类思路》中学习过。你先通过下面这道练习测试一下，看看自己是否已经掌握了这个方法。

  • 主观原因

    • 饮食不健康：经常吃油炸食品、甜品等高热量食物，不吃水果蔬菜 → 可能造成肥胖、内脏负担
    • 缺乏运动：久坐不起、出行只坐车等 → 可能造成关节疼痛、近视等
    • 不良生活习惯：吸烟、喝酒、熬夜等 → 可能造成免疫力下降、内脏负担
  • 客观原因

    • 生活压力大：

      • 社会工作和学习生活节奏普遍较快 → 无法保持健康作息 → 可能造成亚健康
      • 工作、学习、社交压力大 → 可能造成心理问题

    • 环境污染：现代城市空气污染严重 → 可能造成呼吸道疾病

    • 外界诱惑多：现代人生活水平上升 → 食物选择多、容易暴食 → 可能造成肥胖

    • 遗传：肥胖、近视等疾病可能会遗传 → 子女容易得这种病

*这里有一个小细节提醒一下。注意老师在写上述逻辑链条时，用到了“可能造成”这样的表达。这样做是为了避免话说得太过绝对，因为我们列出的原因并不是一定会导致后面的问题。这种表达称之为“模糊语”（hedging），是雅思写作中的一种高分技巧。老师建议你在写大作文的时候，要论证逻辑关系时，也可以视情况用上模糊语，不要写得太过绝对。

提出措施：个人和政府

我们已经在犯罪类思路中学习过，讨论措施的时候，可以对照前面的原因给出解决方案。在健康类话题中，最容易想到的措施可能就是“养成健康的生活方式”，但光说这一点会显得有点空。这时，我们就可以用分情况讨论的方法，按照不同的主体来提出措施。比如，我们可以按个人和政府来讨论：

  • 针对主观原因

    • 针对饮食不健康

      • 个人应该养成健康均衡的饮食习惯、少摄入垃圾食品
      • 政府应该多支持和鼓励健康食品的研发和售卖

    • 针对缺少锻炼

      • 个人需要加强体育锻炼、适当走路或骑车出行
      • 政府应该提倡全民运动，多修建体育设施、举办体育活动

    • 针对不良生活习惯

      • 个人应该戒除吸烟、喝酒等不良习惯
      • 政府应该限制烟酒的购买，比如禁止未成年人购买、数量限购等

  • 针对客观原因

    • 针对生活节奏快压力大

      • 个人可以多发展兴趣爱好，比如进行运动、冥想、做手工等解压活动
      • 政府应该保障劳动权益，杜绝高强度加班现象

    • 针对环境污染

      • 个人可以购买空气净化器、佩戴防雾霾口罩
      • 政府应该进行环境治理，提高空气质量；用法律法规限制企业在生产时的废料污染

    • 针对外界诱惑多

      • 个人应该谨慎选择食物
      • 政府可以加强营养学知识的普及

    • 针对遗传

      • 个人应该改善生活习惯，防止疾病进一步演化
      • 针对某些疾病，政府可以发展医疗水平，开展预防工作

总的来说，如果你拿到一道健康类的题，要求你分析原因和措施，那么就尝试用到我们刚才说的，分析主观和客观原因的方法去发散；发散出了原因，措施也就水到渠成了。再上一个台阶来看，你需要学会把主观（内在）和客观（外在）原因这个分析框架灵活运用在各类话题上，这也是一直在强调的

考点1：文化艺术的价值/作用

这些具体的场景，我们可以发散出文化艺术分别能满足个人和社会的哪些需求，由此将文化艺术的价值大致分为以下这几类：

对个人

实用价值

  • 提升个人能力：如创造力、想象力、审美水准等 → 这些能力有助于提升个人认知水平和综合素养，进而帮助个人更好地完成学习或工作任务，或在实际工作中创造价值

非实用价值

  • 情感宣泄：文艺作品能够表达情感和思想 → 创作或接触文艺作品 → 个人能与作品共鸣，进行情感宣泄
  • 提供娱乐消遣：参与文艺活动 → 能够缓解压力，愉悦身心

  • 美学享受：艺术作品能够提供美学享受 → 让人们享受审美体验 → 增添生活乐趣

对社会

实用价值

  • 促进经济发展：文化艺术产业繁荣发展 → 提供就业机会、促进消费 → 增加经济效益
  • 促进社会和谐发展：文化艺术作品能够提供审美和精神享受 → 增强公民幸福感和社会凝聚力 → 促进社会和谐发展

非实用价值

  • 传承传统文化：文化艺术是文化遗产的重要载体 → 能够保护和传承本土文化

  • 促进文化创新：文化艺术能促进不同文化间的交流和理解 → 有助于增进文化多样性，促进文化创新

  • 塑造国家形象：文化艺术能够展示一个国家的独特文化 → 有助于塑造良好的国际形象
